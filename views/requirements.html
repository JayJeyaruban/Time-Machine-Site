Requirements



Initial Requirements

  • Redesign of current GUI to be more user friendly (making use of current logos and themes)
  • User Logon feature which either makes use of the current network logon credentials or has a password reset functionality in order allows users to access the website
  • Ability to add permissions to certain users to limit sections of the website that they can access
  • Functionality to allow users with the correct permissions to be able to set up groups showing a department's structure with the ability to assign roles i.e. engineer/team manager/reporter/administrator
  • Functionality to generate overtime reports based on the information selected by users using simple filters, and the ability to send these monthly to users on request if they have the correct permissions
  • Functionality to generate the special claims form (again with the ability to send these monthly to users with appropriate permissions) in the necessary format
  • Functionality to be able to integrate with the current ESS system used to determine employee pay
